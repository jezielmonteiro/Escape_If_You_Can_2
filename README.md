<h2>🕹️ Sobre o jogo</h2>
<p><strong>Escape If U Can 2</strong> é um jogo do tipo plataforma, contendo todas as mecânicas abordadas no material disponibilizado na plataforma da disciplina. Essa é a sequência do <a href="https://github.com/jezielmonteiro/Escape_If_U_Can">jogo anterior</a>, que contém novas adições e aprimoramentos.</p>
